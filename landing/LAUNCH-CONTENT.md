## Chrome Web Store Resubmission Pack

Use this section for Chrome Web Store metadata only (not social copy).

### 1) Summary (short description)

```
Capture the current tab and style screenshots with gradients, crop tools, and PNG export.
```

### 2) Detailed description

```
FramedShot lets you capture the visible area of your current tab, then edit the screenshot in a built-in canvas editor.

What you can do:
- Capture the current tab from the toolbar button or context menu
- Add gradient or solid backgrounds
- Adjust angle, padding, shadow, and corner roundness
- Resize the image area and choose aspect ratio presets (Auto, 1:1, 16:9, 4:5)
- Crop the screenshot before export
- Copy to clipboard or download as PNG
- Optionally save your editor settings locally for future captures

Permissions used:
- activeTab: capture the visible area of the tab you trigger from
- contextMenus: add the "Frame this Shot" right-click action
- storage: store the current screenshot and optional saved settings locally
```

### 3) Screenshot/media checklist (to avoid "irrelevant info")

- Show the extension editor UI and controls.
- Show real before/after screenshot examples created by the extension.
- Show crop flow and export actions (copy/download) in the extension UI.
- Keep captions tied to actual features shown in each screenshot.
- Do not use promotional-only media text like `Add to Chrome` in screenshots/video thumbnails.
- Do not use unrelated banners, badges, or claims that are not visible in product behavior.
